# 🔥 DELIKUSH – Ropa Oversize

Página web oficial de **DELIKUSH**, una marca de ropa oversize con estilo urbano, streetwear y cultura kush/hip hop.

---

## ✅ Funcionalidades Implementadas

- **Navbar fija** con efecto blur y scroll activo
- **Hero section** animada con tipografía bold y formas flotantes de colores
- **Franja de características** con información de envíos y beneficios
- **Colección de 6 productos** (Remera, Buzo Hoodie, Pantalón Baggy, Jogger, Set Completo, Musculosa) con botones directos a WhatsApp
- **Sección de pedidos** con guía de 3 pasos y formulario completo
- **Formulario de pedido** que genera mensaje automático para WhatsApp con: nombre, prenda, talle y color
- **Botón flotante de WhatsApp** animado con tooltip
- **Sección "La Vibe Delikush"** con descripción de la marca
- **Sección de contacto** con cards interactivas
- **Footer** con links y botón de WhatsApp
- **Diseño 100% responsive** para mobile, tablet y desktop
- **Animaciones de entrada** al hacer scroll (IntersectionObserver)
- **Menú mobile** con animación hamburguesa
- **Notificaciones toast** de feedback al usuario
